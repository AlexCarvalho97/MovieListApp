## Tech stack

- Kotlin
- Jetpack Compose
- Coroutines for asynchronous
- Dagger Hilt for depencency injection 
- Retrofit 
- Coil
- Architecture: 
  - MVVM Architecture (View - DataBinding - ViewModel - Model)
